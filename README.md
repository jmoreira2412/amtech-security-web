# AM TECH Security — Sitio web informativo

Sitio web oficial de **AM TECH Security** (Panamá): seguridad física digital, infraestructura tecnológica y ciberseguridad.

**Protegemos | Conectamos | Aseguramos**

🌐 **Sitio en vivo:** https://jmoreira2412.github.io/amtech-security-web/

## Estructura

```
index.html        Página completa (HTML + CSS + JS en un solo archivo)
assets/           Logos e identidad visual
```

No requiere instalación ni compilación: es un sitio estático puro.

## Cómo editar

1. Edita `index.html` (textos, servicios, datos de contacto).
2. Haz commit y push a la rama `main`.
3. GitHub Pages publica el cambio automáticamente en 1–2 minutos.

### Ver en local

Abre `index.html` directamente en el navegador, o sirve la carpeta:

```bash
python3 -m http.server 8000
```
